Why Anacron
-----

![][demo]

We used a **RokSprocket** module with the **Lists** layout to make up this area of the front page. You will find the settings used in our demo below.

### Details

![][demo2]

| Option           | Setting         |  
| :--------------- | :-------------- |  
| Title            | `Why Anacron`   |  
| Show Title       | Show            |  
| Access           | Public          |  
| Position         | mainbottom-b    |  
| Status           | Published       |  
| Content Provider | Simple          |  
| Type             | Lists           |
