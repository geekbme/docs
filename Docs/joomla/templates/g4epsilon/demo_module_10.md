Links
-----

![][demo]

We used a **RokSprocket** module with the **Tabs** layout to make up this area of the front page. You will find the settings used in our demo below.

| :--------------- | :---------------------- |  
| Title            | `Links`                 |  
| Show Title       | Show                    |  
| Access           | Public                  |  
| Position         | footer-c                |  
| Status           | Published               |  
| Content Provider | Simple                  |  
| Type             | Tabs                    |
