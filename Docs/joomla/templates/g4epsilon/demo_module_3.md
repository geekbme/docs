Portfolio
-----

![][demo]

We used a **RokSprocket** module with the **Mosaic** layout to make up this area of the front page. You will find the settings used in our demo below.

### Details

![][demo2]

| Option           | Setting     |  
| :--------------- | :---------- |  
| Title            | `Portfolio` |  
| Show Title       | Show        |  
| Access           | Public      |  
| Position         | feature-a   |  
| Status           | Published   |  
| Content Provider | Simple      |  
| Type             | Mosaic      |
